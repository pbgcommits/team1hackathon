## Installation

To run on your machine:

Install the files with
```shell
git clone https://github.com/pbgcommits/team1hackathon.git 
```
Navigate into the directory:
```shell
cd team1hackathon 
```

Install dependencies:
```shell
pip3 install -r requirements.txt
```

Open the website:
```shell
flask run 
```
